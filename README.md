# Where in the world?

Where in the world is a project based on REST Countries API.

### Prerequisites:

yarn or npm

### To set this projetct locally:

Install dependencies:
```
yarn install
```

Build project:
```
yarn build-dev
```

or to production:
```
yarn build
```

Start project:
```
yarn start
```

Access localhost:8080 on your favorite browser
