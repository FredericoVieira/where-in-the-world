# Where in the world?

Where in the world is a project based on [rest countries API](https://restcountries.com/)

### Prerequisites:

- yarn
- node v12.4.0

### To set this project locally:

Install dependencies:
```
yarn install
```

Build project:
```
yarn build-dev
```

or to production:
```
yarn build
```

Start project:
```
yarn start
```

Access localhost:8080 on your favorite browser
